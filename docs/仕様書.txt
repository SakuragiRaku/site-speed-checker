Webサイト表示速度チェッカー 仕様書


アプリ名
  日本語: Webサイト表示速度チェッカー
  英語: Site Speed Checker
  フォルダ名: site-speed-checker


概要
  URLを入力すると、Google PageSpeed Insights APIを使って
  Webサイトの表示速度とパフォーマンススコアを取得・表示するツール。
  モバイルとデスクトップ両方の結果を比較でき、改善ポイントを確認できる。


対象ユーザー
  Web開発者、ブロガー、サイト運営者、SEO担当者。


主な機能

  URL入力と計測
    URLを入力して計測ボタンを押すと、PageSpeed Insights APIにリクエスト。
    モバイルとデスクトップの両方を同時に計測。

  スコア表示
    パフォーマンススコア (0~100) を円形ゲージで表示。
      90~100: 緑 (良好)
      50~89: オレンジ (改善の余地あり)
      0~49: 赤 (要改善)
    Core Web Vitalsの各指標を表示。
      LCP (Largest Contentful Paint): 最大コンテンツの表示時間
      FID (First Input Delay): 初回入力遅延
      CLS (Cumulative Layout Shift): レイアウトのずれ

  改善提案
    APIから返される改善項目 (Opportunities) を一覧表示。
    各項目の影響度と推定節約時間を表示。
    問題の詳細説明と対処法を表示。

  モバイル vs デスクトップ比較
    モバイルとデスクトップのスコアを横並びで比較表示。
    どちらが弱いかを可視化。

  計測履歴
    過去に計測したURLと結果をLocalStorageに保存。
    同じURLの過去のスコアと比較して改善度を追跡。
    日付順の一覧表示。

  お気に入りURL
    よく計測するURLをお気に入り登録してワンタップで再計測。


画面構成
  ホーム画面: URL入力フォーム、お気に入りURL一覧、最近の計測履歴
  結果画面: スコアゲージ (モバイル/デスクトップ)、Core Web Vitals、改善提案一覧
  履歴画面: 過去の計測結果一覧、スコア推移グラフ
  設定画面: テーマ切替


技術スタック
  HTML / CSS / JavaScript
  PageSpeed Insights API (パフォーマンス計測)
  LocalStorage (データ永続化)
  Canvas or SVG (スコアゲージ、グラフ描画)


API設定
  PageSpeed Insights APIはAPIキーなしでも利用可能 (レート制限あり)。
  APIキーを設定すると制限が緩和される。
  APIキーの入力欄を設定画面に用意する (任意)。


データ保存
  LocalStorageに以下を保存する。
    計測結果 (URL、スコア、Core Web Vitals、改善項目)
    計測日時
    お気に入りURL
    APIキー (任意)
    テーマ設定


テーマ
